Nested variables defined directly with controllers

1st Controller

First Name: {{firstName}}

2nd Controller inside 1st

First Name form inside 1st controller: {{firstName}}
last name: (new) {{lastName}}
Full Name: {{getFullName}}

3rd Controller inside 2nd and 1st

First Name form inside 1st controller: {{firstName}}
middle name: (new) {{middleName}}
last name: (from 2nd controller) {{$parent.lastName}}
last name: (redefined) {{lastName}}
Full Name: (redefined) {{getFullName()}}


Nested variables defined inside object with controllers

1st Controller

First Name: {{firstModelObj.firstName}}

2nd Controller inside 1st

First Name form inside 1st controller: {{firstModelObj.firstName}}
last name: (new) {{secondModelObj.lastName}}
Full Name: {{getFullName}}

3rd Controller inside 2nd and 1st

First Name form inside 1st controller: {{firstModelObj.firstName}}
middle name: (new) {{thirdModelObj.middleName}}
last name: (from 2nd controller) {{secondModelObj.lastName}}
last name: (redefined) {{thirdModelObj.lastName}}
Full Name: (redefined) {{getFullName()}}
